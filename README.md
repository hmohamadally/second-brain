# Second Brain — AI Engineering Command Center

Base de connaissances cumulative pour l'ingénierie logicielle augmentée par IA.

## Structure

| Dossier | Contenu |
|---------|---------|
| `adr/` | Architecture Decision Records |
| `patterns/` | Solutions réutilisables |
| `prompts/` | Prompts efficaces (Copilot, Hermes) |
| `workflows/` | Séquences de développement standardisées |
| `debugging/` | Historique incidents/résolutions |
| `retrospectives/` | Analyses post-développement |
| `architecture/` | Diagrammes et specs d'architecture |
| `projects/` | Contexte par projet |
| `snippets/` | Code réutilisable |
| `ai-learnings/` | Apprentissages IA |
| `task-history/` | Historique des tâches exécutées |
| `tasks/` | Tâches actives (Task Protocol) |

## Conventions

- Un fichier = une unité de connaissance
- Frontmatter YAML obligatoire (type, tags, date)
- Liens internes via `[[wikilinks]]`
- Timeline en bas de page (append-only)
- Compiled truth en haut (réécrit si nouvelle info)
